# recipe_book

A lightweight recipe book web project for organizing and browsing recipes in a simple, approachable interface.

## Overview
This project focuses on clean frontend structure and basic web application organization.  
It is ideal for practicing component layout, content hierarchy, and small-scale app UX.

### Key Features
- Recipe listing and presentation
- Simple navigation and page flow
- Beginner-friendly code structure

### Tech Stack
- HTML
- CSS
- JavaScript

## Setup

### Prerequisites
- A modern web browser
- (Optional) VS Code + Live Server extension

### Installation
```bash
git clone https://github.com/r-kyle-kennedy/recipe_book.git
cd recipe_book
```

## Usage Examples

### Run locally
Open `index.html` directly in your browser, or run with Live Server.

### Typical usage
1. Browse available recipes.
2. Select a recipe to view details.
3. Iterate on layout/content to expand functionality.

## Notes
Future enhancements could include search, categories, and persistent storage.
